# DevOps-Learning-Resources
A collection of free resources to learn DevOps methodology and tools that were very helpful in my career.
# 🐧Linux Administration
## ▶️ Courses
- Red Hat System Administration I on [Mahara-Tech](https://maharatech.gov.eg/course/view.php?id=2115) (Highly comprehensive course for RHCSA Certificate)
- Red Hat System Administration II on [Mahara-Tech](https://maharatech.gov.eg/course/view.php?id=2205) (Highly comprehensive course for RHCSA Certificate)
## 📖 Books / Blogs
- [Linux Journey](https://labex.io/linuxjourney) (A short but very good introduction to linux)
- How Linux Works, 3rd Edition by Brian Ward [Book](https://www.oreilly.com/library/view/how-linux-works/9781098128913/) (Useful for learning the 'Why' of Linux/Unix systems)
- UNIX and Linux System Administration Handbook, 5th Edition by Trent R. Hein, Evi Nemeth, Garth Snyder, Ben Whaley, Dan Mackin [Book](https://www.oreilly.com/library/view/unix-and-linux/9780134278308/) (A comprehensive Handbook for linux/unix system administration tasks with a greater focus on legacy systems)
## 💻 Hands On / Labs
- [Bandit](https://overthewire.org/wargames/bandit/) (A capture-the-flag (CTF) style game that teaches you linux)
- [SadServers](https://sadservers.com/scenarios/topic/linux-bash) (A website full of troubleshooting scenarios for linux and DevOps in general)
- KodeKloud Linux Administration [Labs](https://kodekloud.com/studio/labs) (An execllent learning platform for cloud and DevOps that has free hands on labs)
